## 一、迭代背景

### 1.1 背景与目标
在 Phase 1 完成基础记录能力后，我们发现知识工作者面临两个新的核心痛点：
1.  **多项目并行焦虑**：无法在一条时间轴上清晰管理多个并行项目（如白板、数据分析、杂事）。
2.  **AI 协作可视化**：大量工作开始委托给 AI（如自动生成报表），这些任务不应该占用人的"物理精力"，但在时间轴上需要体现。

### 1.2 迭代策略
为了保持开发节奏，我们将需求拆分为两个小阶段：
*   **Phase 1.5 (UI Framework)**：构建 **分屏时间轴 (Split-Pane Timeline)**，解决多项目并行问题。
*   **Phase 1.6 (AI Data)**：引入 **AI 机器人属性**，解决人机协作可视化问题。

## 二、Phase 1.5：分屏时间轴 (Split-Pane Timeline)

### 2.1 核心需求
允许用户将中央时间轴区域拆分为多个平行的垂直 **泳道 (Swimlanes)**，每个泳道代表一个特定的项目上下文。

### 2.2 用户故事
| 角色 | 故事                                     | 验收标准                                        |
| ---- | ---------------------------------------- | ----------------------------------------------- |
| 用户 | 我想同时查看"白板项目"和"常规工作"的安排 | 点击 "+" 按钮，增加一列时间轴，并选择项目过滤   |
| 用户 | 我想让"白板项目"列宽一点，方便操作       | 拖拽列与列之间的分割线，调整宽度                |
| 用户 | 我想在不同项目间移动任务                 | 将任务卡片从 Sidebar 拖入任意列，自动关联该项目 |

### 2.3 交互规范
*   **默认视图**：单列模式（Focus (General)），与 Phase 1 一致。
*   **添加列**：时间轴 Header 区域新增 `Split View` 按钮或 `+ 列` 按钮。
*   **列头 (Column Header)**：
    *   显示项目名称（如 `Project: Whiteboard`）。
    *   **列配置**：提供 `Maximize`（最大化该列）和 `Close`（关闭该列）按钮。
*   **分割线 (Divider)**：
    *   鼠标悬停显示 `Col-Resize` 光标。
    *   支持左右拖拽调整列宽。

## 三、Phase 1.6：AI 机器人属性

### 3.1 核心需求
在时间轴上区分 **"人做的任务"** 和 **"AI 跑的任务"**。AI 任务应具有独特的视觉样式，暗示其"不消耗人的物理时间"。

### 3.2 用户故事
| 角色 | 故事                               | 验收标准                                        |
| ---- | ---------------------------------- | ----------------------------------------------- |
| 用户 | 我想一眼看出哪些任务是 AI 自动跑的 | AI 任务卡片有独特的金属感/渐变背景，并有 🤖 图标 |
| 用户 | 我想标记一个任务为 AI 执行         | 创建/编辑任务时，选择执行者为 `AI_Auto`         |

### 3.3 UI/UX 规范
*   **AI 任务卡片样式**：
    *   **图标**：标题前显示 🤖 (Robot Icon)。
    *   **背景**：区别于普通任务的白底蓝色条，使用 **银灰色渐变 (Metallic Gradient)** 或 **微光效果**。
    *   **边框**：无边框或科技感颜色的微细边框。
*   **交互**：点击 AI 任务，可查看其运行状态（Mockup: `Running`, `Done`）。

## 四、数据结构变更

### 4.1 tasks 表扩展
新增字段以支持 AI 属性：

```sql
ALTER TABLE tasks ADD COLUMN executor VARCHAR(20) DEFAULT 'human'; 
-- values: 'human', 'ai_auto', 'ai_assisted'

ALTER TABLE tasks ADD COLUMN project_id UUID; 
-- 关联项目，用于分屏过滤
```
