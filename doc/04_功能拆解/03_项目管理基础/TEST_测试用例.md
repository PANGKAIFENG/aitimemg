# TEST: 项目管理基础 - 测试用例

## 1. 项目 CRUD

| 用例     | 操作                         | 预期结果                           |
| :------- | :--------------------------- | :--------------------------------- |
| 创建项目 | 点击新建，输入名称，选择颜色 | 项目出现在列表中                   |
| 编辑项目 | 点击编辑，修改名称           | 列表和绑定泳道同步更新             |
| 归档项目 | 点击归档                     | 项目移入"已归档"分组，关联任务保留 |

---

## 2. 任务-项目绑定

| 用例     | 操作               | 预期结果                   |
| :------- | :----------------- | :------------------------- |
| 关联项目 | 任务详情选择项目   | 任务出现在对应项目的泳道中 |
| 取消关联 | 任务详情选择"无"   | 任务仅在主轨道可见         |
| 拖入泳道 | 拖拽任务到项目泳道 | 任务自动关联该项目         |

---

## 3. 泳道与项目联动

| 用例     | 操作                   | 预期结果               |
| :------- | :--------------------- | :--------------------- |
| 绑定项目 | 泳道 Header 选择项目   | 泳道过滤显示该项目任务 |
| 关闭泳道 | 点击关闭按钮           | 项目和任务关联不变     |
| 重新打开 | 新建泳道并绑定同一项目 | 之前关联的任务仍然显示 |
