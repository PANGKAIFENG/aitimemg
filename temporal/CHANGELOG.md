# 更新日志 (CHANGELOG)

本文件记录 Temporal 项目的版本发布历史。

格式遵循 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)。

---

## [0.5.2] - 2026-02-02
### 修复
- **Crash**: 修复了未配置 Supabase 环境变量导致生产环境白屏崩溃的问题。现在支持无 Supabase 模式运行。

## [0.5.1] - 2026-02-02
### 修复
- **UI**: 修复了“任务详情弹窗”高度过高超出屏幕的问题，现在支持内部滚动。
- **UI**: 修复了“AI 协同”任务在窄屏下图标被挤压看不见的问题。

## [1.4.0] - 2026-01-07

### 优化
- **CommandBar 草稿持久化**：输入内容自动保存到 localStorage，关闭后再打开可恢复
- **标签插入位置**：标签现在插入到光标位置，而不是总是追加到末尾
- **DailyTray 布局**："今日任务" 标题移到虚线框外部
- **拖拽阴影修复**：拖拽任务同时滚动时不再闪烁，使用 `visibility:hidden` 替代 `opacity`
- **时间轴重叠布局**：实现 Google Calendar 风格的列分配算法，重叠任务正确并列显示
- **状态指示器**：右上角改为显示 "任务完成进度 + 时间占用"，替代原有 CAPACITY 百分比

### 技术变更
- `taskService.ts`：新增 `getTimeSlotOccupancy()` 和 `getTaskProgress()` 函数
- `Timeline.tsx`：重写 `getTaskPosition` 算法
- `Header.tsx`：更新 props 和显示逻辑

## [1.3.5] - 2026-01-05

### 新功能
- **日期选择**：顶部导航栏支持点击日期弹出日历选择器，方便跨天/月跳转。

## [1.3.4] - 2026-01-05

### 优化
- **UI 细节**：将任务卡片左侧色条改为直角矩形，去除圆角。
- **视觉降噪**：移除任务标题中【标签】的特殊高亮样式，回归普通文本显示。
- **隐形手柄**：隐藏卡片底部的拖拽手柄视觉（横线），仅保留鼠标交互区域，进一步简化界面。

## [1.3.3] - 2026-01-05

### 优化
- **UI 风格**：优化任务卡片样式，移除全背景色，改为左侧色条区分任务类型，视觉更清爽。
- **视觉干扰**：减少多任务堆叠时的色彩干扰，帮助用户专注任务内容。

## [1.3.1] - 2026-01-05

### 优化
- **DailyTray (待排区)**：
  - 新增展开/收起功能，默认收起（高度 128px）。
  - 添加展开箭头图标，优化空间占用，防止遮挡。
- **Sidebar (侧边栏)**：
  - 修复收起状态下内容未隐藏导致的视觉重叠问题。
  - 优化收起动画和透明度过渡。

## [1.3.0] - 2026-01-05

### 新增
- **延期任务功能**：自动识别前一天及更早未完成的已排班任务
  - 在当天顶部待排区域显示延期任务，带橙色主题和「延期 X 天」标签
  - 支持直接勾选完成或点击放弃按钮快速处理
  - 延期任务区固定最大高度，支持滚动查看
  - 拖拽延期任务到时间轴可重新排班，原位置自动消失

### 技术变更
- `taskService.ts`：新增 `getOverdueTasks()` 和 `getOverdueDays()` 函数
- `DailyTray.tsx`：支持延期任务分区展示
- `DraggableTrayItem.tsx`：新增延期标签、放弃按钮等 UI 元素
- `index.css`：新增 `--color-overdue` 和 `--color-overdue-bg` 颜色变量

### 修复
- **界面遮挡**：修复 DailyTray 内容过多时遮挡下方时间轴的问题，现已改为固定高度并支持滚动
- **云端数据类型**：修复云端 API 返回的 `is_all_day` 为字符串 `"false"` 而非布尔值 `false`，导致所有任务被错误识别为全天任务的问题

## [1.2.0] - 2026-01-05

### 优化
- **任务加载性能**：采用「本地优先 + 后台同步」策略，首屏可交互时间从 ~5s 降至 <500ms
- **API 合并**：新增 `/api/sync` 接口，合并 tasks 和 schedules 请求，减少网络往返
- **定时保活**：配置 FC 定时触发器（每 5 分钟），消除冷启动延迟

### 修复
- **日期切换闪烁**：修复切换日期时显示上一个日期任务的闪烁问题，现在会立即清除状态并仅从本地加载

---

## [1.1.0] - 2026-01-04

### 新增
- **右键菜单**：任务卡片右键支持「复制任务」和「删除任务」
- **富文本备注**：备注输入框支持粘贴图片，图文混排
- **富文本备注**：备注输入框支持粘贴图片，图文混排
- **日程卡片 UI**：全面升级为 Pastel 柔和色系风格，移除重边框，统一复选框样式
- **删除弹窗**：使用 Glassmorphism 风格的自定义确认弹窗替代原生提示框

### 修复
- 修复侧边栏拖拽任务到时间轴时不可见问题
- 修复顶部待办区拖拽时出现残影问题
- 修复数据刷新后丢失问题
- 修复拖放精度问题（现支持半小时粒度）
